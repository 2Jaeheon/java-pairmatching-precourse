# 기능 요구사항 정리

# 목적: 페어 매칭

# 객체식별
Crew: 크루 개인의 정보를 관리
Level(enum): 레벨을 관리
Course(enum): 코스를 관리
Pair: 페어 매칭한 결과를 저장 및 관리

# 전체 시나리오
0. 파일로부터 데이터 읽어와서 데이터 저장
   - Crew 에 저장하면 됨
1. 기능 선택 (페어 매칭, 페어 조회, 페어 초기화, 종료)
2. 페어 매칭
   - 과정과 미션 출력
     - 템플릿 사용
   - 매칭하고자 하는 과정, 레벨, 미션 입력
     - `InputView`, `OutputView`
   - 매칭 수행 결과 출력
     - OutputView로 Pair 객체 넘겨서 
3. 페어 조회
   - 과정과 미션 출력
   - 조회를 위한 과정, 레벨, 미션 입력
   - 페어 결과 출력
4. 페어 초기화
   - 초기화
5. 종료
   - 종료
